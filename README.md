# luxury-housing-analysis
Real estate data analysis using Python, SQL, and Power BI

     Luxury Housing Sales Analysis – Bengaluru

    Overview

This project analyzes luxury housing data using Python, SQL, and Power BI to understand market trends and generate useful business insights.

    Tools Used
Python (Pandas, NumPy)
SQL
Power BI
Google Colab

    Project Pipeline
Data cleaning using Python
Data stored and analyzed using SQL
Dashboard created using Power BI

    Key Insights
3BHK houses have highest demand
Amenities improve booking success
Some micro-markets perform better than others
Certain builders dominate revenue
